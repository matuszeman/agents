# Table Management

When managing tables in markdown:
- All columns must be properly aligned with consistent padding
- Column separators (`|`) must be vertically aligned
- Header separator (`---`) width must match or exceed the widest cell in that column
- Use consistent spacing: pad cells to align all `|` characters vertically
- Verify alignment by checking that all `|` separators form straight vertical lines
